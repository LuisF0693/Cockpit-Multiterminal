# Story 14.2: Shell de layout (header, sidebar, telemetria, rodapé de sessões)

## Status

InReview

## Story

**As a** fundador,
**I want** a estrutura de tela do mockup (header compacto, sidebar em seções, painel direito, rodapé de cards),
**so that** o app tenha a MESMA anatomia da referência, só que com meus dados reais.

## Acceptance Criteria

1. Header 42px: marca "MEU **COCKPIT**" (ciano), abas de view, `v{versão} · N sessões · ● daemon` (dot colorido pelo estado real), pill de zoom (movido da toolbar 13.2) e "+ novo terminal" (FR48).
2. Sidebar esquerda 240px em seções maiúsculas: PROJETO (ativo com cor/caminho/badge git-native + `⎇ branch` real da 13.3), NOVO AGENTE (adapters reais com descrição do catálogo 13.4 — clicar cria terminal), PROJETOS (dot+nome, + cria), ARQUIVOS (árvore real), APP & SISTEMA (views secundárias REAIS: Timeline/Learnings/Agentes/Configurações) e rodapé de build (versão real, SEM custos) (FR48).
3. Painel direito 230px: TELEMETRIA + STATUS (card de decisões pendentes real, clicável → master) + EVENTOS (timeline real, cores por tipo, poll de 10s) (FR48).
4. Rodapé 88px: cards das sessões ATIVAS reais (dot de status, nome, cwd, tempo no status, chip do adapter — SEM custo); clicar foca o terminal (FR48).
5. Nenhuma funcionalidade removida — views re-hospedadas (secundárias na sidebar), criação por adapter na sidebar/catálogo.
