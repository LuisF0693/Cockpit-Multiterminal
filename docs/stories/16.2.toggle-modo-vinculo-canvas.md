# Story 16.2: Toggle manual↔auto na etiqueta do vínculo no canvas

## Status

InReview

## Story

**As a** fundador,
**I want** alternar o modo do vínculo clicando na etiqueta (`manual`/`auto`) direto no canvas,
**so that** eu não precise remover e recriar o vínculo na sessão master só pra trocar o modo.

## Acceptance Criteria

1. A etiqueta central do bezier (14.4) vira clicável (`data-no-pan`, cursor pointer, tooltip indicando o modo destino) — clique alterna `manual`↔`auto`.
2. Novo IPC `terminalLink.setMode` (Zod na borda) + `TerminalLinkManager.setMode` persistindo via `StateStore.updateTerminalLinkMode` (memory + sqlite; sem migração — coluna `mode` já existe desde a 9.1).
3. Evento `updated` no `TerminalLinkEvent` (push Main → renderer) — o espelho substitui in-place, preservando posição/cor posicional da 14.4.
4. `setMode` é no-op silencioso pra id inexistente ou modo igual (nenhum evento emitido).

## Tasks / Subtasks

- [x] Task 3 — IPC `terminalLink.setMode` (canal, schema, handler, preload, `CockpitApi`) (AC2)
