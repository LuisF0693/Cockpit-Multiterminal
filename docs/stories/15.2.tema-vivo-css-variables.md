## Story

**As a** fundador,
**I want** trocar tema, cor de destaque e fontes com aplicação IMEDIATA (sem reiniciar),
**so that** a Aparência das Configurações (15.3) funcione como na referência OmniRift.

## Acceptance Criteria

1. Tokens expostos como CSS variables (`--ck-*`) no `:root`; o objeto `theme` mantém a MESMA forma com valores `var(--ck-*)` — nenhum componente muda a leitura (FR55).
2. Presets: Multerminal Escuro (default, valores EXATOS do Épico 14), Multerminal Claro, Meia-noite; 7 cores de destaque (par primária/clara); fontes texto/mono selecionáveis (stacks seguras).
3. Concatenações de alpha sobre valores de tema migradas pra `color-mix()` (4 pontos: minimapa ×2, board ×2) — nada quebra com var().
4. xterm re-tematiza AO VIVO: `TerminalView` consome os dados CRUS via `getActiveTheme()`/`subscribeTheme()` e muta `term.options.theme`/`fontFamily` na troca.
5. Persistência: `themePreset`/`accentColor`/`fontText`/`fontMono` nas settings (FR46); App compõe+aplica no boot.
