# Squad Design Expert

Os melhores designers e especialistas em UX do mundo, reunidos para criar experiências digitais excepcionais.

| Agente | Especialidade | Papel |
|--------|--------------|-------|

## Fluxo de Trabalho

```
1. Don Norman define estratégia e arquitetura de informação
   ↓
2. Joanna Wiebe escreve copy e proposta de valor
   ↓
3. Josef Müller-Brockmann define tipografia e grid
   ↓
4. Jessica Walsh cria o design visual
   ↓
5. Jony Ive especifica interações e animações
   ↓
6. Dieter Rams aprova ou devolve com lista de correções
```

## Filosofias do Squad

- **Don Norman:** "Design existe para servir às pessoas — não para impressionar designers."
- **Jessica Walsh:** "Designers têm medo de cor. Eu tenho medo de bege."
- **Josef Müller-Brockmann:** "O grid não limita o design — liberta o designer do caos."
- **Joanna Wiebe:** "Não escreva copy. Colete copy — da boca dos seus clientes."
- **Jony Ive:** "A simplicidade não é a ausência de desordem — é a conquista da ordem perfeita."
- **Dieter Rams:** "Weniger, aber besser. Menos, mas melhor."

## Critérios de Aprovação

Todo design aprovado pelo squad deve ter:
- Score Dieter Rams ≥ 7.5/10
- Score de Usabilidade Don Norman ≥ 7/10
- Sem CTAs genéricos (validado por Joanna Wiebe)
- Todos os estados de componentes especificados (validado por Jony Ive)
- Contraste WCAG AA em toda tipografia (validado por Josef)
